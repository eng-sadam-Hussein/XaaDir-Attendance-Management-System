# XaaDir App - React JS + Tailwind CSS Frontend

This is the completed Tailwind CSS version of the frontend for **XaaDir Attendance Management System**.

## Stack

- React JS
- Vite
- Tailwind CSS
- React Router DOM
- Context API
- Axios
- React Toastify
- Lucide React Icons

## Features

- Professional modern UI using Tailwind CSS
- Blue, green, and orange XaaDir brand colors
- Login page
- Admin dashboard
- Teacher dashboard
- Role-based sidebar
- Protected routes
- Users CRUD
- Classes CRUD
- Subjects CRUD
- Students CRUD
- Attendance marking and CRUD
- Reports
- Search
- Validation
- Toast notifications
- Responsive layout
- Connected to ASP.NET Core Web API backend

## Backend API

Default backend API:

```text
http://localhost:5168/api
```

## Run

Start backend first:

```bash
cd /d/Projects/XaaDir-Attendance-Management-System/backend/XaaDirApi
dotnet run
```

Start frontend:

```bash
cd /d/Projects/XaaDir-Attendance-Management-System/frontend/xaadir-client
npm install
npm run dev
```

Open:

```text
http://localhost:5173
```
